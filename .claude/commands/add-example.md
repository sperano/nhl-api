Create an example in `examples/` demonstrating {{feature}}:

1. Create `examples/{{example_name}}.rs`
2. Write clear, documented code showing:
   - How to create a client
   - How to use the {{feature}} API
   - How to handle errors
   - How to work with the response data
3. Add helpful comments explaining each step
4. Test the example: `cargo run --example {{example_name}}`
5. Add a note about the example to README.md if appropriate

Examples should be production-ready code that users can copy.
